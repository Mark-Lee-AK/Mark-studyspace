###内存图
####基本类型
```
int i = 8;
```
在栈内开辟空间存放变量名和值

####数组引用类型
```
//定义数组
int[] arr = {1,2,3}
//输出数组和数组的各个元素
System.out.println(arr); // -> 0x88
System.out.println(arr[0]); // -> 1
System.out.println(arr[1]); // -> 2
System.out.println(arr[2]); // -> 3
```
* 先声明压栈 arr
* 在堆内存中分配空间，位置假设为0x88开始
* 申请三个连续的内存地址，数组的空间是连续的，并设置各个元素初始值为0
* 然后执行{1,2,3,}进行数组初始化，分别替换每个空间中的值
* 最后把栈中的值指向引用的地址0x88
